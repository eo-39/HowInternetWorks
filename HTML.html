HTML
By Ebinehita Olumese

The last topics we are going to discuss are HTML, W3C, and CSS. HTML stands Hyper Text Mark-Up Language and created by Tim Bernes Lee. HTML is the language used to tell a web browser how a page should look. This language quickly became the web standard for building pages. How does Html work? Well, a webpage is simply an HTML file, made up of many different tags. Tags or elements are hidden keyword that tells a page how it looks.

Tim Bernes Lee in 1993, founded the W3C organization. The W3C is an international community made up of several hundred members and IT professionals whose made objective is to develop protocols and guidelines to ensure the growth of the internet. Finally, CSS is a simple design language used to make web pages more presentable. CSS stands for Cascading Style Sheets and is far more superior at styling your webpage compared to Html. CSS can optimize your page view on multiple devices.
