DNS
By Ebinehita Olumese

The next topic we will be discussing is DNS, which stands for Domain Name System. DNS is designed to make the internet accessible to human beings. DNS translates IP address from numbers to letter.

How does the DNS process work? First, your web browser or device performs a DNS lookup, which is just finding the IP address of a domain. Then, that request is received by the DNS resolver, whose responsible for finding the correct IP address for the hostname. Once, resolver reacher DNS server it goes through a TLD( Top Level Domain) for example .com to name server of a specific domain. When it reaches said Domain, it acquires the Ip address and returns it back to the DNS client. Finally, the client can connect to the correct server through IP address
