HTTP and HTTPS
By Ebinehita Olumese

The next topic we are going to discuss is HTTP and HTTPS protocols. HyperText Transfer Protocol was created by Tim Bernes and his team around 1991. This protocol basically led to the creation of the web browser. HTTP is the language used to communicate between web browsers and servers!

HTTPS is the secure vision of HTTP, your modern-day web browser takes security very seriously. Any site not secured we be labeled as such and provide a warning to the user that the site is not secure. HTTPS secures your information through the use of encryption, which means hiding information. So, encryption can range from 128 bits to 64 bits and this technique makes it very challenging for anyone to decipher if they intercept the communication.
